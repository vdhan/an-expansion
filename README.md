# An's expansion mod for VCMI
Rebalance skills, heroes, creatures, etc.

## Structure:
- Friendly Creature
- Long Live Peasant
- Modern Font
- Neutral Upgrade

- Heroes:
  - Double Movement
  - Resource
  - Skeleton Warrior

- Skills:
  - Archery
  - Armorer
  - Estates
  - Leadership
  - Luck
  - Navigation
  - Offence
  - Resistance
  - Scholar
  - Scouting
  - Sorcery
  - Transport: Combines Logistics and Pathfinding
  - Warfare: Combines Artillery, First Aid and Ballistics
  - Learning:
    - Eagle Eye

- War Machine:
  - War Machine 2
  - Cannon
