# An's expansion mod
Đọc hướng dẫn Tiếng Việt trong trò chơi.

Rebalance skills, heroes, creatures, etc.

## Structure:
- Friendly Creature
- Long Live Peasant
- Modern Font
- Neutral Upgrade
- Light
- Double Movement:
  - Sea

- Heroes:
  - Resource
  - Skeleton Warrior
  - Gold

- Skills:
  - Archery
  - Armorer
  - Estates
  - Leadership
  - Luck
  - Offence
  - Scholar
  - Sorcery
  - Transport: Combines Logistics and Pathfinding
  - Warfare: Combines Artillery, First Aid and Ballistics
  - Learning:
    - Eagle Eye

  - Scouting:
    - Mithril
    
  - Navigation:
    - Battle Mage

- War Machine:
  - War Machine 2
  - Cannon
